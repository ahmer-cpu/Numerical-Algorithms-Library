# Numerical Algorithms Library

A collection of numerical algorithms and computational methods.

## Repository Structure

- `Dense matrix routines/` - Core dense matrix operations and utilities.
- `Factorizations/` - Matrix factorization methods.
- `Linear system solvers/` - Algorithms for solving linear systems.
- `Non-linear system solvers/` - Methods for solving nonlinear systems.
- `Numerical quadrature/` - Numerical integration routines.
- `Polynomial Interpolation/` - Polynomial interpolation algorithms.
- `Splines/` - Spline interpolation and related routines.
