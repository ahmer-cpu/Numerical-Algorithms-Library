# Numerical Algorithms Library

A collection of numerical algorithms and computational methods.

## Project Website

- Website: https://ahmernadeem.com/research/numerical-monte-carlo-library.html
- This page contains a project overview and detailed programming reports describing the code, tasks, experiments, results (including figures), and conclusions.

## Repository Structure

- `Dense matrix routines/` - Core dense matrix operations and utilities. **Language:** C++ and Jupyter Notebook.
- `Factorizations/` - Matrix factorization methods. **Language:** Python and Jupyter Notebook.
- `Linear system solvers/` - Algorithms for solving linear systems. **Language:** Jupyter Notebook (Python).
- `Non-linear system solvers/` - Methods for solving nonlinear systems. **Language:** Jupyter Notebook (Python).
- `Numerical quadrature/` - Numerical integration routines. **Language:** MATLAB (`.m`, `.mlx`).
- `Polynomial Interpolation/` - Polynomial interpolation algorithms. **Language:** MATLAB (`.m`, `.mlx`).
- `Splines/` - Spline interpolation and related routines. **Language:** MATLAB (`.m`, `.mlx`).
